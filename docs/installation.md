# Installation

## Stable release

To install Customer Churn Analytics, run this command in your terminal:

```sh
uv add customer_churn_analytics
```

Or if you prefer to use `pip`:

```sh
pip install customer_churn_analytics
```

## From source

The source files for Customer Churn Analytics can be downloaded from the [Github repo](https://github.com/Pratyush-Basu/customer_churn_analytics).

You can either clone the public repository:

```sh
git clone git://github.com/Pratyush-Basu/customer_churn_analytics
```

Or download the [tarball](https://github.com/Pratyush-Basu/customer_churn_analytics/tarball/master):

```sh
curl -OJL https://github.com/Pratyush-Basu/customer_churn_analytics/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd customer_churn_analytics
uv pip install .
```
